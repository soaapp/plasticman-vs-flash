# Hero photos

Drop the six fighter images here so the character cards and the dossier gallery
pick them up. Vite serves this folder at `/assets/…`.

Expected filenames (3 per fighter):

```
flash-1.png    flash-2.png    flash-3.png
plastic-1.png  plastic-2.png  plastic-3.png
```

`.png`, `.jpg`, `.jpeg`, and `.webp` all work — the app tries each extension and
falls back to the comic emblem if an image is missing. Roughly square / portrait
crops look best (cards show them ~16:10, the dossier gallery a bit taller).
